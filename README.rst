AGVTOOL port for python
====================================

Automating Version and Build Numbers

Installation
~~~~~~~~~~~~
pip3 install agvtool_port

Usage
~~~~~~~~~~~~

agvtool new-version -all 1722
agvtool new-marketing-version 1.0.0
agvtool what-version
agvtool what-marketing-version
